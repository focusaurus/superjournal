This app is using the following stack:

* node.js for a server side runtime and web server
* coffeescript for the primary implementation language.
    * Everything that can be coffeescript is coffeescript.
* express.js for web routing and templating
* jade for web templating
* stylus CSS preprocessor:
    * <http://learnboost.github.com/stylus/docs/selectors.html>
* zombie.js for end to end testing
* jasmine for BDD style testing
* jasbin for command line unit tests
* commonjs assert for unit testing mechanics
* jsdom for server side DOM manipulation (experimentally)
* jquery for server and client side DOM manipulation
* See <package.json> for the specific npm dependencies and versions
* custom jQuery UI theme
